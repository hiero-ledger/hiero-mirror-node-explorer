# hiero-ledger/hiero-mirror-node-explorer

Visual Explorer for the Hiero DLT.

## Project Structure

The explorer is currently a full-client application built with Vue.js and using the [Hiero Mirror Node](https://github.com/hiero-ledger/hiero-mirror-node) REST API.

The code of the explorer is located in the `_frontend` directory.

Setup instructions are located in the [README.md](https://github.com/hiero-ledger/hiero-mirror-node-explorer/blob/main/_frontend/README.md) file.
